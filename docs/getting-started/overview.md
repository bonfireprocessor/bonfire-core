# Getting Started Overview

This section is the recommended entry point for new contributors.

## Goals

After working through these pages, you should be able to:

- create or refresh the local Python environment,
- install the required Python dependencies,
- ensure a suitable RISC-V toolchain is available,
- run the project's main regression suite.

## Recommended path

1. Read [Prerequisites](prerequisites.md)
2. Follow [Quick Start](quick-start.md)
3. Continue with the hardware and workflow sections depending on what you want to change
