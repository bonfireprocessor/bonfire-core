# Bonfire Core Documentation

Bonfire Core is a configurable RISC-V core written in MyHDL, together with SoC wrappers, test infrastructure, and small software test programs.

This documentation site is a first draft built from the repository's existing Markdown documentation. The current focus is on a clear structure:

- **Getting Started** first
- then **Hardware**-oriented topics
- then **Workflows**
- then **Software**

## What you will find here

- how to set up the repository and run the test suite,
- how the core and SoC-level hardware pieces fit together,
- the main developer workflows,
- the software and compliance test assets shipped in the repo.
