# Core Test Programs

The `code/` tree contains small RISC-V assembly test programs used by the core integration tests.

## Main location

- `code/core-tests/`

## Generated artifacts

The existing documentation describes these generated outputs under `code/build/core-tests/`:

- `*.elf`
- `*.hex`
- `*.lst`
- `*.sym`

## Purpose

These programs are used to exercise the complete core in the integration testbench.
